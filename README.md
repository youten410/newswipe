# ニュースワイプ
初めて作ったアプリ。
Tinderライクにニュースが見れたら面白そうと思ったのがきっかけで作ったアプリ。
マッチングのロジックまでは開発できませんでしたが、Flutterでのライトモード、ダークモードや、APIによるデータの受信などを学びながら作りました。
